Orion's Dungeon

HTML, CSS, jQuery, git

I decided to create a traditional roguelike game with pixel art graphics. Writing this game was a bit tricky in the beginning, once I got used to the logic it became fairly simple as I had to repeat most of it for the different scenarios. For the logic I used jQuery and stored state in the location of the tiles in the DOM. Used keyboard events for player movement and timers for random enemy movement and attack.
